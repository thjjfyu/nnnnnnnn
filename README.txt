1) Создай бота в BotFather и получи токен.
2) Скопируй .env.example в .env и вставь BOT_TOKEN.
   ADMIN_IDS можно оставить пустым или указать свой id (через запятую).
3) Установи зависимости:
   python -m pip install -r requirements.txt
4) Запусти:
   python bot.py
5) Создай закрытый чат, добавь туда бота.
6) В этом чате напиши /set_target (от админа).
7) В личке боту напиши /start и пройди анкету.
Результат придёт в закрытый чат.
